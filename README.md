# Telemedicine-Platform
AI-powered smart healthcare platform using microservices, enabling appointment booking, video consultations, medical record management, and AI-based health insights. Integrates secure payments, notifications, and scalable APIs, following cloud-native principles with Docker and Kubernetes deployment.

## Overview
A **microservices-based smart healthcare platform** that allows patients to book doctor appointments, attend video consultations, and manage medical records.  

The system also integrates **AI-based symptom analysis**, **secure payments**, and **real-time notifications**, following distributed system principles.

---

## Core Features
- Patient registration & profile management  
- Doctor management & scheduling  
- Appointment booking & tracking  
- Video consultations (Telemedicine)  
- Online payment integration  
- Notifications (Email / SMS / WhatsApp)  
- AI symptom checker *(optional)*  

---

## Architecture
Built using **Microservices Architecture**, including:
- Patient Service  
- Doctor Service  
- Appointment Service  
- Telemedicine Service  
- Payment Service  
- Notification Service  
- AI Service  

---

## Tech Stack

**Backend**
- Node.js  
- Express.js  
- MongoDB  

**Frontend**
- React.js  

**DevOps**
- Docker  
- Kubernetes  

---

## User Roles
- **Patient** – Book appointments, attend consultations  
- **Doctor** – Manage availability, conduct sessions  
- **Admin** – Manage system and users
